# HTML + JavaScript Starter

## Quick Start
1. Open `public/index.html` in your browser, or use a simple HTTP server.
2. Edit files under `src/` for your app logic and styles.

## Structure
- `public/` static files (entry HTML)
- `src/` source code (CSS, JS, assets)
- `tests/` unit test placeholders

## Dev Tips
- Keep modules small and focused.
- Use `services/` for external I/O (storage, network).
